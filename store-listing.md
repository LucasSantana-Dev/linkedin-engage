## Short Description (132 chars max)

## Detailed Description
LinkedIn Engage automates your LinkedIn networking workflow with intelligent safety features built in.

**Connection Automation**
- Search-based profile discovery with geo and network filters
- Customizable connection notes with name personalization
- Template library (EN/PT-BR) for different career levels
- Duplicate detection prevents re-sending invitations

**Feed Engagement**
- Auto-react and comment on feed posts
- Content-aware NLP classifies posts by topic
- Concept extraction for contextual comments
- Skip keywords to avoid irrelevant content

**Company Following**
- Batch follow companies from search results
- Rotation across multiple company lists
- Scheduled follow campaigns

**Connection Nurturing**
- Auto-track new connections for 7-day engagement window
- Visit nurture targets' recent activity to stay visible
- 3 engagements per connection with 12-hour cooldown

**Smart Scheduling**
- Analytics-driven optimal time windows
- Day-of-week weighting based on engagement history
- Acceptance rate tracking by hour and template

**Safety First**
- Hourly, daily, and weekly rate limits per mode
- Human-like behavioral fingerprinting (gaussian delays)
- Session profiles with unique timing patterns
- Automatic backoff on LinkedIn security challenges
- CAPTCHA/challenge detection with graceful stop

**Analytics Dashboard**
- Engagement stats by mode, category, and reaction
- Template acceptance rates
- Hour-of-day performance heatmap
- ROI tracking for connection campaigns

All data stored locally — no external servers or tracking.
